The Idea

Inherit traits from parents — but also learn from experience.
🧬

Dual chromosomes

A topology chromosome encodes hidden-layer sizes; a parameter chromosome stores weights organized by layer, so feature detectors are inherited hierarchically.

🎯

Backprop as a side channel

Each newborn agent imitates the current best "role model" via backprop before entering the world — no fixed supervised task required, so it applies to any RL problem.

🏁

Faster convergence

Against a pure-GA baseline, LBNE reaches higher fitness per generation by combining exploration (GA) with gradient refinement (BP).

Fitness Function

Reward speed, punish zigzagging, eliminate stalling.
Score_t = Score_(t-1) + Speed × 2 − |Direction| × 3 − 1
